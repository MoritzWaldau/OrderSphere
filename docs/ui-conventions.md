# UI Conventions

The binding UI, theming, MudBlazor, and CSS reference for OrderSphere lives in
[`.github/copilot-instructions.md`](../.github/copilot-instructions.md).

The design direction is **"Flat & Focused"** (indigo `#3A4DD1`, Space Grotesk display,
JetBrains Mono numerics, light-only). The canonical implementations are
`src/OrderSphere.Web/Services/ThemeState.cs` (palette, typography, layout) and
`src/OrderSphere.Web/wwwroot/app.css` (design tokens and component classes).

This file is intentionally a pointer to avoid two diverging style guides.
